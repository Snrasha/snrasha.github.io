Sprites tutorial for users
  • Require Texture Replacement Plugin
  • Launch the game one time for let the plugin setup the folder.
  • Unzip the content on the Super_Fantasy_Kingdom\ModAssets\Replacements\
  • Monsters need to be Super_Fantasy_Kingdom\ModAssets\Replacements\Monsters
  • Heroes/Guardian need to be Super_Fantasy_Kingdom\ModAssets\Replacements\Heroes

Sprites tutorial for spriters
Guardians
A guardian has:
  • [guardian_name]_SpriteSheet for animation
  • [guardian_name]_Grid for the icon on the hero selection
  • [guardian_name]_Icon for the portrait
Heroes
A hero has:
  • [hero_name]_SpriteSheet for animation
  • [hero_name]_Icon for the portrait
For their evo, check the wiki for their evolution name.
Monsters
A monster has:
  • [monster_name]_SpriteSheet
  • [monster_name]_EvoSpriteSheet
  • Idle frames is unused
